# Engineering Mathematics Toolkit

**A Python-based engineering mathematics application combining numerical methods, calculus, vector analysis, and symbolic computation in a unified graphical interface.**

## Overview

This project was developed for **CSAI 101 – Fundamentals of Programming and
Computer Science**. It implements a set of mathematical tools commonly used
in engineering and scientific computing — root-finding algorithms, vector
operations, and calculus utilities — integrated into a desktop GUI built with
CustomTkinter.

## Features

### Newton-Raphson Module
- Single-variable root approximation
- Multiple root detection
- Convergence control
- Numerical error handling

### Vector Mathematics Module
- Vector magnitude
- Dot product
- Cross product
- Angle between vectors
- Angle with coordinate axes
- 2D and 3D vector support

### Calculus Utilities Module
- Definite integration
- Area between two functions
- Tangent line calculation
- Normal line calculation
- Symbolic function evaluation

### Graphical User Interface
- Interactive mathematical calculations
- Tab-based navigation
- Real-time result visualization
- User-friendly input handling

## Learning Outcomes

This project developed practical experience in algorithm design and
mathematical modeling, applied through the implementation of numerical and
symbolic methods in Python. It also involved structuring a multi-module
codebase around object-oriented principles and building a functional GUI to
expose these tools interactively, reinforcing both scientific computing and
software design skills.
